# 🧾 选你妹

别纠结，打印出来就按它办。

选你妹是一个用来治选择困难的小网页。把几个选项写进去，点一下按钮，它会随机替你选一个。界面是复古小票风格，像把纠结打印成一张收据。

## 🤯 为什么做它

有些选择不值得开会，也不值得在脑子里循环播放。

- ❌ 吃什么、喝什么、先做哪个，想半天没有结果。
- ❌ 候选项其实都差不多，但你还是想要一个“决定”。
- ❌ 用转盘太重，用表格太傻，只想快速抽一个。

这个网页不负责帮你分析人生，它只负责把选择打印出来。

## 🎯 它适合什么场景

- 今天吃什么。
- 周末去哪玩。
- 几个任务先做哪个。
- 抽一个候选标题、候选方案、候选灵感。
- 手机浏览器里临时做个小决定。

## 💡 核心功能

- ✅ 自己输入候选项。
- ✅ 随机抽出一个结果。
- ✅ 保存最近抽过的结果。
- ✅ 支持手机浏览器打开。
- ✅ 支持 PWA，部署到 HTTPS 后可以添加到手机主屏幕。

## ⚙️ 怎么用

每行写一个选项，比如：

```text
火锅
烧烤
麻辣烫
```

然后点 `PRINT MY CHOICE`。

## 🛠️ 本地打开

这个项目不需要安装依赖。

在项目目录里运行：

```bash
python3 -m http.server 4173
```

然后打开：

```text
http://127.0.0.1:4173/
```

## 🚀 部署

这是一个纯静态网页，直接部署整个目录就行。

适合部署到：

- Netlify
- CloudBase 静态网站托管
- Webify
- GitHub Pages

入口文件是：

```text
index.html
```

不需要构建命令。
